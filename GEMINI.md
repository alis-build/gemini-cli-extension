# Alis Build Extension Context

You are working with Alis Build through the `alis-build` MCP server and the `alis-build-agent` remote sub-agent.

When the user asks about Alis Build, prefer Alis Build MCP tools before guessing. Use the tools to inspect the active workspace, product, environment, build status, logs, deploy status, ideas, and related service context.

Use `@alis-build-agent` for delegated Alis Build workflows that benefit from the hosted Alis Build agent runtime, especially broad workspace review, build/deploy guidance, and implementation planning.

Do not trigger rebuilds or deploys unless the user asks.

Keep responses concise and action-oriented.
